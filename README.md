## Sobre o projeto

O **Travelgram** é uma página de perfil focada em apresentação visual de conteúdo de viagem. O projeto trabalha hierarquia visual, grid de imagens, espaçamento, responsividade e acabamento de interface.

Esse projeto demonstra fundamentos essenciais para front-end júnior: estrutura semântica, estilização consistente e adaptação para diferentes telas.

---

## Funcionalidades

- Layout de perfil inspirado em rede social
- Galeria visual de viagens
- Organização clara de conteúdo
- Estilização com CSS
- Estrutura responsiva
- Interface simples, limpa e objetiva
